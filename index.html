BÀI 18: TẠO TRANG WEB & THAY ĐỔI GIAO DIỆN

1. Luyện tập 1

✦ Câu hỏi:

Tạo trang dang_ki.html chứa biểu mẫu đăng kí câu lạc bộ và bổ sung liên kết đến trang này ở cuối tất cả các trang.

Bài làm:

• Tạo tệp dang_ki.html → tạo biểu mẫu bằng <form>.

<form>
  <label>Họ và tên:</label>
  <input type="text">

  <label>Email:</label>
  <input type="email">

  <label>Lớp:</label>
  <input type="text">

  <button>Đăng kí</button>
</form>

• Thêm liên kết cuối mỗi trang:

<footer>
  <a href="dang_ki.html">Đăng kí câu lạc bộ</a>
</footer>

2. Luyện tập 2

✦ Câu hỏi:

Thay đổi màu sắc và định dạng phông chữ khi di chuột qua các vùng.

Bài làm:

.vung {
  font-size: 14px;
  color: #333;
  font-family: Arial, sans-serif;
}

.vung:hover {
  font-size: 16px;
  color: red;
  font-weight: bold;
}

3. Vận dụng

✦ Câu hỏi:

Đưa ra một thiết kế mới cho website đã tạo và áp dụng toàn bộ định dạng mới.

Bài làm:

• Tạo file style.css. • Viết CSS thiết kế mới:

.banner {
  background: url("../assets/img/bg-masthead.jpg") no-repeat center;
  background-size: cover;
  padding: 12rem 0;
  color: darkred;
  text-align: center;
}

.slogan {
  background-color: #f8f9fa;
  text-align: center;
  padding: 7rem 3rem;
}

.row {
  display: flex;
  flex-wrap: wrap;
  padding: 0 3rem;
}

.block_3 {
  flex: 0 0 33.33%;
}

.block_3 h3 {
  font-size: 24px;
  font-weight: bold;
}

.text-content {
  font-size: 16px;
